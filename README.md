# Bài tập lớn Thiết kế Hệ thống Nhúng - EE3003

A mini console game's project from EE3003 - Ho Chi Minh City Of Technology - National University of HCM City. Using 1 Led Matrix 8x8 as the display which is can make amount of sound effect like moving sound, eating sound, sound when its gameover or game already begin. At the Phase 1 of Project, we gonna make The Snake Game as well as using some matrix algorithm to control our snake, moving up or down, left or right, become longer whenever eating food, we also applied the rand()'s fundamental to make food appearing randomly on the Led Matrix. If I have more time and be excited in this project, we gonna moving on to the Phase 2, create and import a menu game, which is it have more options to play more game if we mind, I might thinking about The Tetris...
